# Online Retail Data Analysis

## Overview
This project analyzes an online retail store dataset to understand customer purchasing patterns, product demand, and sales trends.

## Key Analysis
- Monthly sales trends
- Top selling products
- Country-wise sales distribution
- Customer purchasing behavior
- Revenue insights

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Outcome
The project provides business insights that help in inventory planning, marketing strategy, and customer targeting.
